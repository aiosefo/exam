Simple jQuery activity

Complete the following tasks.

1. Hide/Show. Use jQuery to hide each square when you click on it. Then click on button to show them again.

2. Toggle. Use jQuery to toggle the square below.

3. Discussion. What is the difference between hide()/show() and toggle()? When would you use one over the other?

4. Apply some animation to the square below. It is up to you what you want to do. You can change its position, size, color, etc.

5. Use the alert method to display an alert message when clicking on the button below.

5. Use the addClass() method to add a separate class to the ".question". Style it differently (eg. change font-size, font color). You will need to add that class to the inline style section above.
